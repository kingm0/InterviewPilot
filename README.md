## Getting Started

### Prerequisites

- Python 3.7+
- Jupyter Notebook
- (Optional) Node.js for frontend components

### Installation

1. **Clone the repository**
    ```bash
    git clone https://github.com/kingm0/InterviewPilot.git
    cd InterviewPilot
    ```

2. **Install Python dependencies**
    ```bash
    pip install -r requirements.txt
    ```

3. **Start Jupyter Notebook**
    ```bash
    jupyter notebook
    ```

4. **(Optional) Set up frontend**
    ```bash
    # If there is a frontend directory
    cd frontend
    npm install
    npm start
    ```

## Usage

- Launch Jupyter Notebook and open any notebook to begin practicing.
- Track your progress with built-in analytics.
- (If available) Use the web interface for an enhanced experience.
